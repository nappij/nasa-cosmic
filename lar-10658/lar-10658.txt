 FAST MARS RELAY COMMUNICATION LINK

( MARTIN MARIETTA CORP. )

     This program evaluates the communications link between the Viking Or-
biter and Lander vehicles. The program calculates the trajectory of the Or-
biter and Lander simultaneously. Using data from both vehicles, this pro-
gram calculates communication geometry.

     This program has been implemented on the CDC 6400 and 6500 computers.
